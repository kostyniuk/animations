# Resources

Trusted sources for grounding lessons. Never trust parametric knowledge — cite these.

## Knowledge
- [MDN — Using CSS animations](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_animations/Using_CSS_animations) — keyframes as a function of time. Used in Lesson 01.
- [MDN — requestAnimationFrame](https://developer.mozilla.org/en-US/docs/Web/API/Window/requestAnimationFrame) — the frame loop primitive. For Lesson 02.
- [MDN — Canvas API / drawing basics](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API/Tutorial) — 2D context, drawImage, transforms.
- [Gaffer On Games — Integration Basics](https://gafferongames.com/post/integration_basics/) — why motion is `pos += vel·dt` and what dt-independence means. Used in Lesson 01.
- [web.dev — CSS Houdini](https://web.dev/articles/css-houdini) — how far CSS-driven custom paint can go (and its limits). Referenced in Lesson 01.

## Skills / tools
- [Motion (motion.dev)](https://motion.dev/) — JS animation library for the "state, few elements" tier.
