# Baseline: comfortable with JS, new to canvas

Established at workspace setup (2026-06-18). The learner writes JS and CSS comfortably (including CSS animations/transitions) but has not done canvas rendering or per-frame animation loops.

Implications:
- Lean on JS intuition; don't re-teach closures, events, or CSS basics.
- Treat canvas + the `requestAnimationFrame` loop + `dt`-based integration as the genuinely new material.
- Mission is "understand the techniques", so prioritise mental models and live demos over build-from-scratch projects.
